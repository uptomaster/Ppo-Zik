## 🌟 서비스 소개

### 배경

한국인의 평균 화장실 체류 시간은 **12분**. 성인의 **77%**가 화장실에서 스마트폰을 사용하고 있으며, 치질은 20·30대 항문 질환 **발생률 1위**를 차지합니다. 3분 이상 변기에 앉아 있으면 항문 혈관에 압박이 가해져 치질 위험이 급격히 상승합니다.

**뽀직이**는 귀여운 캐릭터와 실시간 타이머를 통해 자연스럽게 건강한 화장실 습관을 형성하도록 도와주는 웹 앱입니다. 단순한 타이머를 넘어 AI 기반 식단 추천과 배변 패턴 분석을 통해 **개인 맞춤형 장 건강 코치** 역할을 합니다.

### 핵심 가치

| 🎯 재미 | 💚 건강 | 🤖 스마트 |
|:---:|:---:|:---:|
| 귀여운 뽀직이 캐릭터로 습관 형성을 즐겁게 | 과학적 근거 기반의 장 건강 가이드 | Claude AI 기반 개인화 식단 추천 |

## ⚙️ 핵심 기능

### 1. ⏱️ 스마트 타이머
변기에 착석하는 순간 타이머가 시작되며, 경과 시간에 따라 6단계로 위험도를 시각화합니다.

| 단계 | 시간 | 상태 | 색상 |
|------|------|------|------|
| SAFE | 0 ~ 1분 | 쾌적해요 😊 | 초록 `#00E5A0` |
| OKAY | 1 ~ 2분 | 괜찮아요 🙂 | 노랑 `#FFD60A` |
| CAUTION | 2 ~ 3분 | 슬슬 위험해요 😐 | 주황 `#FF9A3C` |
| WARNING | 3 ~ 5분 | 위험해요! 😰 | 짙은 주황 `#FF6B00` |
| DANGER | 5 ~ 7분 | 치질 위험! 🚨 | 핑크 `#FF2D78` |
| CRITICAL | 7분 이상 | 즉시 일어나! ☠️ | 퍼플 `#9D4EDD` |

### 2. 💩 뽀직이 캐릭터
- 6단계 감정 표현 (행복 → 약간 걱정 → 무표정 → 걱정 → 패닉 → 사망)
- 시간 경과에 따른 몸 색상 변화
- 땀방울, 증기 이펙트 애니메이션
- 위험 단계 도달 시 떨림(shake) 애니메이션

### 3. 📊 배변 기록 & 캘린더
배변을 마친 후 물을 내리는 애니메이션과 함께 기록을 저장합니다.

**Bristol Scale 7단계 분류**
```
1형: 딱딱한 덩어리   →  변비 의심
2형: 소시지형 딱딱   →  변비 경향
3형: 균열 있는 소시지 →  정상 범위
4형: 매끈한 소시지   →  이상적 ✅
5형: 부드러운 덩어리  →  정상 범위
6형: 흐릿한 조각    →  설사 경향
7형: 완전 액체      →  설사 의심
```

**캘린더 기능**
- 날짜별 배변 횟수 시각화 (안전 🟢 / 주의 🟡 / 위험 🔴)
- 날짜 클릭 → 상세 기록 펼침
  - 몇 시에 앉았는지
  - 배변 소요 시간
  - Bristol 형태 배지
  - 변 색상 기록
  - 메모

### 4. 🥗 식사 기록
- 아침 / 점심 / 저녁 / 간식 시간대 구분
- 음식명 직접 입력 또는 빠른 선택 칩
- 음식별 장건강 점수 자동 계산

**음식별 장건강 점수 예시**
```
브로콜리  9점   |  현미밥  8점   |  요거트  8점
바나나    8점   |  오트밀  9점   |  물 2L  10점
라면     1점   |  피자    2점   |  콜라    0점
삼겹살   3점   |  케이크  0점   |  커피    4점
```

### 5. 📅 통계 대시보드
- 7일간 일평균 배변 시간 바 차트
- 총 세션 수 / 평균 시간 / 위험 횟수 / 오늘 횟수

## 🤖 AI 건강 분석 시스템

### 종합 건강 점수 산출
```
종합 점수 = 배변 횟수 점수 (25%)
          + 배변 시간 점수 (30%)
          + 변 형태 점수  (25%)
          + 식단 점수    (20%)
```

**점수 등급**

| 점수 | 등급 | 설명 |
|------|------|------|
| 85 ~ 100 | 🌟 훌륭해요 | 장 건강이 매우 좋은 하루예요 |
| 70 ~ 84  | 👍 좋아요   | 꾸준히 유지하면 더 좋아져요 |
| 50 ~ 69  | 😐 보통     | 식단과 화장실 습관을 개선해봐요 |
| 30 ~ 49  | ⚠️ 주의 필요 | 장 건강에 신경 써야 할 것 같아요 |
| 0 ~ 29   | 🚨 위험     | 오늘 식습관과 배변 패턴을 점검하세요 |

### Claude AI 식단 추천 플로우

```
배변 기록 입력  →  식사 데이터 입력  →  Claude AI 분석
                                            ↓
점수 업데이트  ←  맞춤 식단 추천  ←  통합 분석 결과
```

**AI 추천 포함 내용**
1. 현재 장 건강 상태 평가
2. 오늘 먹으면 좋은 음식 3가지 (이유 포함)
3. 오늘 피해야 할 음식 2가지
4. 내일 추천 식단 메뉴

**API 연동**
```javascript
// Claude claude-sonnet-4-20250514 모델 사용
const response = await fetch("https://api.anthropic.com/v1/messages", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    model: "claude-sonnet-4-20250514",
    max_tokens: 1000,
    messages: [{
      role: "user",
      content: `배변 기록: ${records}회, 평균 시간: ${avgTime}초\n식사: ${meals}\n맞춤 장건강 식단을 추천해주세요.`
    }]
  })
});
```

## 🔔 특별 알림 시스템

### 🫙 0회 — 변비 경고
하루 화장실 기록이 없을 때 발동합니다.

- 하루 1~3회가 건강한 배변 횟수임을 안내
- 물 2L 이상 + 식이섬유 섭취 권고
- **2일 이상 지속 시** 전문의 상담 권고
- 아침 공복 따뜻한 물 마시기 가이드 제공

### 🚨 5회 이상 — 과다배변 경고
하루 5회 이상 배변 시 발동합니다.

- 과민성대장증후군(IBS) 의심 증상 안내
- 자극적·기름진 음식 회피 추천 식단 제공
- **3일 이상 지속 시** 의료기관 방문 강력 권고
- 스트레스·복통 동반 여부 체크 프롬프트

### ✨ 완벽 패턴 — 축하 이벤트
1~3회 배변 + 건강 점수 80점 이상 달성 시 발동합니다.

- 뽀직이 왕관 이모지 축하 배너
- 연속 건강일 카운터(스트릭) 증가
- 주간 베스트 기록 달성 시 특별 배지
- SNS 공유 가능한 리워드 카드 생성 *(Phase 3)*

## 📐 건강 점수 알고리즘

### 배변 횟수 점수 (25%)

| 하루 횟수 | 점수 |
|-----------|------|
| 1회 | 100점 |
| 2~3회 | 95점 |
| 0회 | 20점 |
| 4~5회 | 60점 |
| 6회 이상 | 20점 |

### 배변 시간 점수 (30%)

| 평균 소요 시간 | 점수 |
|--------------|------|
| 60초 미만 | 100점 |
| 60~120초 | 90점 |
| 120~180초 | 70점 |
| 180~300초 | 45점 |
| 300초 이상 | 15점 |

### 변 형태 점수 (25%) — Bristol Scale 기준

| Bristol | 점수 | 설명 |
|---------|------|------|
| 4형 | 100점 | 이상적 |
| 3형 또는 5형 | 80점 | 정상 범위 |
| 2형 또는 6형 | 60점 | 경계 |
| 1형 또는 7형 | 15점 | 이상 의심 |

### 식단 점수 (20%)

```javascript
// 음식별 점수 예시
const FOOD_SCORES = {
  '브로콜리': 9,  '블루베리': 9,  '오트밀': 9,
  '현미밥':   8,  '바나나':   8,  '요거트':  8,
  '삼겹살':   3,  '커피':     4,  '치킨':    5,
  '라면':     1,  '피자':     2,  '콜라':    0,
};

// 오늘 식단 평균 점수 × 10 = 식단 점수
const dietScore = Math.round(
  (todayMeals.reduce((sum, m) => sum + m.score, 0) / todayMeals.length) * 10
);
```
